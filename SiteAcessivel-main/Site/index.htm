Ferramentas Utilizadas
O que fazemos?
Análisamos os Dados

Nossa equipe se dedica à análise de dados no setor de petróleo, com foco em informações fornecidas pela Agência Nacional do Petróleo, Gás Natural e Biocombustíveis (ANP). Utilizamos esses dados para compreender o processo de processamento de petróleo no Brasil, bem como as dinâmicas de exportação e importação.

Conectamos os Pontos

Nosso principal foco é correlacionar esses dois dados para fornecer uma visão completa das atividades relacionadas ao petróleo. Identificamos tanto as sinergias quanto os desafios que afetam o Brasil e ajudamos os usuários a compreender como esses fatores se relacionam e impactam o país.

Dados Acessíveis a Todos

Nosso compromisso é disponibilizar esses dados de maneira clara e compreensível para que qualquer pessoa, independentemente de sua formação, possa acessar e compreender as informações essenciais relacionadas ao setor de petróleo no Brasil.

Acesse o nosso código

Python é uma linguagem de programação amplamente reconhecida por sua flexibilidade e poderosa biblioteca de ferramentas para análise de dados. Utilizamos essa linguagem para processar e analisar cada conjunto de dados de forma detalhada.

Para facilitar nossa análise e colaboração, escolhemos a plataforma Google Colab. Esta plataforma baseada na nuvem nos oferece um ambiente de desenvolvimento integrado para Python, permitindo-nos escrever código, executar análises e compartilhar resultados de forma eficiente.

Após analisar cada conjunto de dados individualmente, nossa equipe se concentra na correlação de dados provenientes de diferentes fontes. Isso envolve a identificação de padrões, tendências e relações entre os dados de processamento de petróleo, exportação e importação. Através do Python e das ferramentas disponíveis no Google Colab, conseguimos realizar análises estatísticas avançadas e visualizações de dados que destacam como esses fatores se relacionam entre si.
